# IRON-GREAT-WALL-WAR-OF-SURVIVAL-p5.js
This project is a 2D side-scrolling shooter built with p5.js as a high school CS30 capstone project. It follows a classic arcade structure with a complete game flow including menu, tutorial, difficulty selection, gameplay, Game Over, continue system. Players win by defeating enough enemies or the final boss.
Web to play: https://dmnzgh.github.io/IRON-GREAT-WALL-WAR-OF-SURVIVAL-p5.js/
